### Diminish
As 7, Half diminish 7 settle to Root Triad 
(Bm7b5 -> C)

As 3
Cmaj7 -> Dm7 -> Em7b5 -> A7 -> Dm7 -> G7 -> Cmaj7

Dm7 -> G7 -> Em7b5 -> Aaug7 -> Dm7 -> G7 -> Cmaj7

As 2
C -> Dm7b5 -> G7 -> C   As Major 
Am -> Bm7b5 -> E7 -> Am   As Minor


Diminish 7 settle to Root minor (Bdim7 -> Cm)
